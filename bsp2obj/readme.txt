GK3 BSP To OBJ Converter
v0.1
http://gk3tools.sourceforge.net/
------------------------

== USAGE ===============
bsp2obj some_bsp_file.bsp [optional_output_file.obj]

If no output file is specified a default filename of default.obj
is used.
