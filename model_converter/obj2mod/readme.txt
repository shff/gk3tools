GK3 OBJ To MOD Converter
v0.0.1
http://gk3tools.sourceforge.net/
------------------------

== USAGE ===============
obj2mod some_obj_file.obj [optional_output_file.mod]

If no output file is specified a default filename of default.mod
is used.
